# Hobies
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/Gityus13/Hobies)

Hobies is a dynamic and interactive website created for the EAS (European Azerbaijan School) 7F class. It serves as a creative platform to showcase students' hobbies and personalities through a visually engaging, animated grid. The site is packed with hidden features, including thematic modes, mini-games, and an integrated AI chatbot.

The live website can be viewed at: **[eashobies.site](https://eashobies.site)**

## Features

*   **Student Showcase:** An animated grid displaying students from class 7F with descriptions of their hobbies and interests.
*   **Secret Interactive Modes:** Unlockable themes that completely transform the website's appearance and add unique interactive elements.
*   **Game Center:** A collection of fun, simple browser-based games including Flappy Bird, a Perfect Circle drawing challenge, and memory games.
*   **AI Chatbot:** An integrated AI assistant ("AliGpt") for users to interact with.
*   **Dynamic UI:** Includes a dark mode toggle, background music controls, and extensive animations using AOS and anime.js.
*   **'Add to Dock' for Mac:** A dedicated guide explaining how macOS users can add the website to their dock for easy access.
*   **Informational Pages:** Features pages with videos about the school, class sponsors, and a contact form.

## Interactive Modes

On the homepage, locate the input field at the top of the screen. Type one of the following secret codes to activate a unique theme:

*   `gta`: Activates a Grand Theft Auto-inspired theme with a custom cursor, sound effects, and HUD elements.
*   `disco`: Transforms the site into a dance party with flashing lights, a spinning disco ball, and animated elements.
*   `barbie`: A pink, glittery theme inspired by Barbie, complete with special cursors and effects.
*   `glitch`: A digital corruption theme with screen tearing, scrambled text, and simulated error messages.
*   `matrix`: A hacker-style theme with a green-on-black terminal aesthetic and a falling code effect.

## Games

The site includes several built-in games for entertainment:
*   **Flappy Bird:** A classic implementation of the popular mobile game.
*   **Perfect Circle Game:** A challenge to test your ability to draw a perfect circle with your mouse.
*   **Games Page:** A collection of smaller games, including a Memory Match card game, a Reaction Time Test, and Rock Paper Scissors.

## Technology Stack

*   **Frontend:** HTML5, CSS3 (with Tailwind CSS), JavaScript
*   **Animation Libraries:** AOS (Animate On Scroll), anime.js
*   **Icons:** Feather Icons, Font Awesome
*   **Hosting:** GitHub Pages
